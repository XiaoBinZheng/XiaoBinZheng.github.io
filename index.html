Markdown语法备忘

Posted on 2016-07-16   |   In Tools   |  

行内HTML

Markdown可以直接嵌入Html的格式语法，例如：

This is a regular paragraph.

<table>
    <tr>
        <td>Foo</td>
    </tr>
</table>

This is another regular paragraph.

效果：

This is a regular paragraph.

Foo

This is another regular paragraph.

段落与换行

一个段落由一个相连接的行句组成，通过一个以上的空行可以切分成不同的段落。

通过输入<br />标签，或者直接在行尾加上两个以上的空白，然后按enter，也可以实现换行的效果。

Markdown更推荐使用引用和多段落的列表格式来排版。

标题

Markdown支持两种标题的语法，Setext和atx形式。

Setext形式是用底线的形式，利用=（最高级标题）和-（第二级标题），例如：

This is an H1
=============

This is an H2
-------------

Atx形式则是在行首插入1到6个#，各对应标题h1到h6，例如：

# This is an H1

## This is an H2

###### This is an H6

区块引言

引言语法，示例：

> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
> consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
> Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.
> 
> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
> id sem consectetuer libero luctus adipiscing.

效果:

This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.

Markdown也允许你只在整个段落的第一行最前面加上>：

> This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

> Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.

效果:

This is a blockquote with two paragraphs. Lorem ipsum dolor sit amet,
consectetuer adipiscing elit. Aliquam hendrerit mi posuere lectus.
Vestibulum enim wisi, viverra nec, fringilla in, laoreet vitae, risus.

Donec sit amet nisl. Aliquam semper ipsum sit amet velit. Suspendisse
id sem consectetuer libero luctus adipiscing.

区块引言可以有级别，根据级别加上不同数量的>: 

> This is the first level of quoting.
>
> > This is nested blockquote.
>
> Back to the first level.

效果:

This is the first level of quoting.

This is nested blockquote.

Back to the first level.

引言的区块内可以 使用其他的Markdown语法，如标题、列表、程序代码区块等。

> ## This is a header.
> 
> 1.   This is the first list item.
> 2.   This is the second list item.
> 
> Here's some example code:
> 
>     return shell_exec("echo $input | $markdown_script");

效果：

This is a header.

  1. This is the first list item.
  2. This is the second list item.

Here’s some example code:

return shell_exec("echo $input | $markdown_script");

列表

Markdown语法支持有序列表和无序列表。

无序列表使用星号、加减或是减号作为列表标记：

*   Red
*   Green
*   Blue

等价于：

+   Red
+   Green
+   Blue

也等价于：

-   Red
-   Green
-   Blue

效果：

  • Red
  • Green
  • Blue

有序列表使用数字加一个英文句点：

1.  Bird
2.  McHale
3.  Parish

效果：

  1. Bird
  2. McHale
  3. Parish

程序代码区块

使用4个空白或是1个tab，示例：

This is a normal paragraph:

    <div>This is a code block.</div>

效果：

This is a normal paragraph:

<div>This is a code block.</div>

行内程序代码区块，可以使用（`）符号将代码包起来，例如：

Use the `printf()` function.

效果：

Use the printf() function.

分割线

示例（省略显示效果）：

* * *

***

*****

- - -

---------------------------------------

链接

示例：

This is [an example](http://example.com/ "Title") inline link.

[This link](http://example.net/) has no title attribute.

效果:

This is an example inline link.

This link has no title attribute.

参考形式的链接使用另外一个方括号接在链接文字的括号后面，而在第二个方括号要填入用以辨识链接的标签，示例：

This is [an example] [id] reference-style link.

接着，在文档的任意位置，定义标签的链接内容：

[id]: http://example.com/  "Optional Title Here"

效果:

This is an example reference-style link.

强调

Markdown使用星号（*）和下划线（_）作为标记强调内容的符号，被*或包围的内容会被<em>标签包围，用两个*或者\，则会被<strong>包围，例如：

*single asterisks*

_single underscores_

**double asterisks**

__double underscores__

效果：

single asterisks

single underscores

double asterisks

double underscores

图片

Markdown使用一种和链接很相似的的语法来标记图片，同样也允许两种样式：行內和参考。

行内图片语法，示例：

![Alt text](/path/to/img.jpg)

![Alt text](/path/to/img.jpg "Optional title")

格式为：

![图片替代文字](图片url "图片title")

参考式图片引用语法（具体参考链接的参考式引用）：

![Alt text][id]
[id]: url/to/image  "Optional title attribute"

天池流行音乐比赛小结

Posted on 2016-07-16   |   In Competitions   |  

竞赛介绍

本次大赛以阿里音乐用户的历史播放数据为基础，期望参赛者可以通过对阿里音乐平台上每个阶段艺人的试听量的预测，挖掘出即将成为潮流的艺人，从而实现对一个时间段内音乐流行趋势的准确把控。

竞赛详情，点击天池大数据竞赛。

第一赛季

时间序列算法

由于数据是具有时序的，目标是预测接下来的两个月的播放量，所以第一想法是用时间序列来处理这个问题。本来打算用Matlab来处理这个问题，看到队友用python的工具包处理数据的过程，不得不说python在数据处理和分析上更快速便捷，于是决定学一下python。

这里推荐《利用Python进行数据分析》这本书。

基本思路是，提取每个艺人6个月里每天的播放量数据，对每个艺人的播放量数据单独绘制自相关图、偏相关图。根据绘制的图像，计算不同的ARMA模型，从中选择最合适的模型，进行残差分析，结果预测等。

由于没有多少时间花在比赛上，所以一开始的目标就是维持进复赛的资格。提交了数据后很长一段时间没有动手。最后第一赛季结束时排到了480多名，勉强进入复赛。

第二赛季

进了复赛后，就只能在平台进行比赛，第一次使用平台带来了很大麻烦。加上到了期末，没有多少时间，最后只是勉强提交了一个结果，感谢队友费了不少心。

回归算法
取得平台资格后，发现平台并没有提供时间序列算法的模型，而且基本上只有靠SQL来解决问题。考虑到平台有回归模型可以直接调用，所以想到了用回归模型来处理这个问题。

基本思路是，提取特征，对每一个艺人建立回归模型。但是遇到了很多问题，最终没有实现。

小结

  1. 学会用python处理和分析数据
  2. 了解了阿里的大数据平台
  3. 多浏览论坛的内容，可以获得很多有用的信息

Day 5

早上7点半，包车前往若尔盖大草原，司机是一个藏族人，我们称呼他为“王叔”（王是他的汉族名的姓，藏族名字太长了，没记住）。王叔很年轻，才四十多，有两个孩子，一个今年刚参加完高考。由于今年他忙着装修自己的房子，所以我们是他今年的第一批客人。

下午10点多，到达了若尔盖大草原，在这里体验了射箭，藏族火锅，骑马，藏服。

图5-1 大草原上的蓝天白云

图 5-3 从左到右，青稞饼、藏族火锅、烤羊肉、酥油茶

下午2点左右，出发前往唐克镇，九曲黄河第一湾景区所在的镇。

晚上6点多左右，到达唐克镇，入住景区门口外的77连锁酒店（由于是淡季，只要100一个晚上）。我们没打算游览景区，只是想第二天早上在这里看日出。王叔还帮我们联系了前几天来这里看日出的侄子，从中得到了一些攻略。

晚上7点左右，在酒店开的餐厅吃饭，由于入住了这家酒店，老板给我们打了8折。

第五天结束。

旅行之四姑凉与山——Day 4

Posted on 2016-06-21   |   In Trips   |  

##Day 4

早上7点半集合，旅舍老板没法载我们过去九寨沟风景区入口。于是，在路口叫了一辆黑车，辛苦4个菇凉挤3个人的位置。

早上8点10分左右，到售票处买九寨沟风景区的门票。据老板说，8点半后来九寨沟的旅游团很多，建议早一点买票。不过，我们8点10分到九寨沟风景区门口的时候，已经有不少旅游团了。

早上8点30分左右，开始游玩九寨沟。

线路是，犀牛海-原始森林-箭竹海-熊猫海-珍珠滩瀑布-诺日朗中心。

图3-1 犀牛海

图3-2 珍珠滩瀑布（西游记取景的地方）

一路上，玩得挺开心。然而，悲剧的事发生了。

下午3点20分左右，到了诺日朗买完明信片和邮票后，准备搭观光车去长海的时候，发现自己的相机包丢了。几经周折后，确定相机包在公交车上，不过车已经到了停车场。

下午5点左右，搭乘路边的车到了停车场，中间又发生了一些小插曲，又到了另一个山上的停车场，终于拿回了相机包。开车送我们去停车场的司机是个本地人，帮了我们不少忙。拿回相机包后，司机问我们要不要去藏族人家那里吃饭，有篝火晚会和藏族表演。可能是因为这个司机帮了我们大忙，再加上我破坏了四菇凉的九寨沟之旅，心理十分内疚，而且菇凉们也有些兴趣，所以我自作主张，请四位菇凉去藏族人家吃饭。

事实证明，我做了一个错误的决定。当我们进去的时候，发现里面就只有我们几个游客，十分冷清。当我们坐下的时候，那个藏族菇凉的表情和表演也让我们觉得不安。吃的是藏族火锅，但是比昨天的火锅差多了，不仅辣还很咸。我们几个人开始觉得自己被坑了，甚至怀疑这里是不是传销地点。直到后面陆续进来了两三个旅游团的人，才稍微有点放心。综合看来，这里估计就是旅游公司忽悠旅游团游客的地方之一。本来就很饿的姑凉们却一点也吃不下，我更内疚了，又坑了四姑凉们一次，顿时对那个司机感到愤懑。思考过后，决定不吃了，郁闷起身走人。

出去外面后，我打了个电话给那个司机（他说吃完饭后送我们回去）。上车的时候，他问我觉得怎么样，我没有好脸色的回他，“你觉得呢”。那个司机估计也听明白了，全程没开口讲一句话。

回到旅舍后，各自回房休息。自己一个人在房间内疚了很久。

晚上9点左右，开始讨论隔天的行程。由于对若尔盖大草原、花湖等地方没什么了解，请来了旅店的老板，帮我们决定明天的行程。最后我们决定第二天包车去若尔盖大草原，老板还帮我们联系了一个司机（老板同学的爸爸，是一个很朴实的人，后面一路上给了我们不少好的建议）。

第四天结束。

事实证明，我做了一个错误的决定。当我们进去的时候，发现里面就只有我们几个游客，十分冷清。当我们坐下的时候，那个藏族姑娘的表情和表演也让我们觉得不安。吃的是藏族火锅，但是比昨天的火锅差多了，不仅辣还很咸。我们几个人开始觉得自己被坑了，甚至怀疑这里是不是传销地点。直到后面陆续进来了两三个旅游团的人，才稍微有点放心。综合看来，这里估计就是旅游公司忽悠旅游团游客的地方之一。本来就很饿的姑娘们却一点也吃不下，我更内疚了，又坑了四姑娘们一次，顿时对那个司机感到愤懑。思考过后，决定不吃了，郁闷起身走人。

出去外面后，我打了个电话给那个司机（他说吃完饭后送我们回去）。上车的时候，他问我觉得怎么样，我没有好脸色的回他，“你觉得呢”。那个司机估计也听明白了，全程没开口讲一句话。

回到旅舍后，各自回房休息。自己一个人在房间内疚了很久。

晚上9点左右，开始讨论隔天的行程。由于对若尔盖大草原、花湖等地方没什么了解，请来了旅店的老板，帮我们决定明天的行程。最后我们决定第二天包车去若尔盖大草原，老板还帮我们联系了一个司机（老板同学的爸爸，是一个很朴实的人，后面一路上给了我们不少好的建议）。

第四天结束。

旅行之四姑娘与山——Day 3

Posted on 2016-06-20   |   In Trips   |  

Day 3

昨晚信誓旦旦说7点钟集合，结果睡过头了（隐约中记得好像闹钟响过，被我顺手关了…）。7点06分被手机震醒。然后花了不到5分钟，洗漱完毕。接着又花了几分钟找房卡（居然被扔到垃圾桶了…）。恩，在一阵慌乱后，终于可以出发了（假装镇定）。本来打算自行过去沟口搭车，没想到老板自己过来问我们用不用载我们过去。

早上7点半左右，有惊无险上了车。沟口车站从九寨沟到黄龙的车，貌似只有7点半这一班，车费是45。在车上，可以直接让乘务员帮我们订黄龙景区的门票，门票是200（学生价100），上去的索道是80。返程时间是下午3点半。

早上9点半左右，在高原休息站停车。休息站的工作人员上车开始给我们普及到高原的注意事项，然后就是推销缓解高原反应的药（恩，终于等到广告了）。不过，我们在来的时候已经吃了红景天了（后面几天根本忘了这回事，也就没再用过了），所以丝毫没理会她们那些略带夸张的宣传。去黄龙的途中，会经过一个叫雪宝顶的地方，最高海拔5588米，停车的地方估计只有海拔4000多米。一下车，明显感觉到温度低了很多。海拔明显有些影响，稍微跑动一下，都会觉得气喘。

图3-1 雪宝顶（远处有雪山）

早上11点左右，到达黄龙景区门口。等乘务员取票后，开始坐索道上去。前半段路的时候，走得太慢了。结果到后面非常赶，记得在下午3点的时候，我们离出口还有3千米。前面路段没什么景色，建议走快点，到了五彩池景点的时候可以稍微慢点欣赏，往后的景点有些类似，容易产生审美疲劳，可以看得快一点。

图3-2 五彩池

下午6点30左右，回到九寨沟。决定去一家叫做如意林卡的餐馆吃饭。价格还可以接受，我们点了藏式火锅、青稞酒、酥油茶、牦牛酸奶、青稞饼。推荐藏式火锅和酥油茶，味道相当不错。耗牛酸奶也挺有特色的，跟广州的酸奶不太一样。

吃完饭后，在附近逛了一下。买了一些干粮。晚上9点左右，旅舍老板打电话过来问我们要回来没，本来打算打车回去的，老板居然送上门来了，果断叫老板送我们回去。不过，老板的车被借了。等老板借车等了好一会（当然还是要谢谢老板）。

回到旅舍后，照旧在我房间吃东西，玩游戏。

第三天结束。

旅行之四姑娘与山——Day 2

Posted on 2016-06-19   |   In Trips   |  

Day 2

估计是昨晚早睡的缘故，早上6点30分左右就醒了。我住的是6人间，刚好昨天全住满了。担心吵醒他们，慢慢爬下床，发现下铺的阿坝藏族小伙已经醒了，但愿不是我的缘故。话说昨晚勇气十足地挑战麻辣火锅，后果就是今早肚子频繁地向我抗议。

洗漱完，整理好行李，在一楼的木桌上，边写游记，边等四姑娘起床。从成都新南门车站到九寨沟有6个班次，在广州时已经提前订好了今天早上8点的车票，昨晚提前去车站取了票，所以时间充裕。

告别了青旅老板，左手拖着箱子，右手提着面包、泡面和八宝粥（后来我把微信群名改成“泡面与面包之旅”的缘故），前往新南门车站坐车。

早上8点10分左右，车出发了。4个姑娘刚好分成两对坐一起，再一次感慨自己有点多余。本期待旁边是一位可人的姑娘的愿望也被一位大叔的出现彻底粉碎。就这样，一路上基本是在睡觉中度过的。中途有停车休息的地方，不建议在这些地方买东西（帽子、水果、干粮等），贵的离谱。更何况九寨沟并不像他们说的一无所有（后面到了才发现这里基本上应有尽有）。

下午6点左右，客车到达九寨沟沟口。顺手到沟口的旅客客运中心取明早去黄龙的车票。在沟口这里，到处都是拉客的黑包车司机，一直纠缠着说个不停。闲着无聊，刚好询问了下他们去若尔盖、花湖那些地方的价格。

取完票，打电话给来之前网上订好的旅舍的老板，让他过来接我们。因为下着雨，只好在旁边店铺的屋檐下等老板过来，过程中那些躲雨的黑车司机又纠缠了一番（烦躁。。。）。大概15分钟后，打电话告知具体位置后，旅舍老板伞也不打（据说是已经丢了n把，放弃治疗了…）跑过来接我们去停车的地方。好吧，第一感觉是这个人靠谱么（事后证明，这个老板挺靠谱的。当然，也挺二的。据说绰号“二哥”）。到了旅舍后，发现旅舍就在沿河的地方，环境不错，没有前台，老板住在其中的一间房，好像就老板一个人打理的样子。据四姑娘说，当时决定选这家酒店是因为它刚开不久，价格便宜，离车站不太远。

旅舍名字叫“西行向北青年旅舍”，老板是一个毕业4年的80后（后来知道是89年的），人很好。旅店的环境还不错，地板和天花板都是木质的，从整体来看应该建了有一段时间了（后面听老板说，这里是他租下来的，还在完善中）。后面吃饭的时候，听到姑娘们在吐槽洗手间的门是没有锁的，然而我住的是单间，并没什么关系（一个人住的好处）。

图 2-1 右边就是老板（借用的图片，左边非本人），中间的画，据老板说是住他旅店的人帮忙画的

老板建议我们去西餐厅吃饭，还建议我们试下牦牛肉披萨。在附近晃悠了一会后，还是决定听从老板的建议。进去看了菜单后，用一句话来形容就是“麻雀虽小，五脏俱全”。除了点了老板说的牦牛肉披萨，还点了水果沙拉等等。看到菜单饮料一栏写着“黯然销魂”、“温柔一刀”，还以为是调酒。于是，和姑娘J决定一人点一种。好在，点菜的小哥听错了。结果只上了一壶（小壶）黯然销魂。庆幸上错的原因是，原来黯然销魂是酿制的木瓜酒，度数大概有50多度（没错，平时我们顶多只喝10度不到的啤酒，竟点了壶50多度的酒）。好吧，自己点的酒跪着也要喝完。好在酒的味道不错，点的几道菜味道也挺不错（不知道是不是因为在成都吃东西留下的阴影，造成了反差，吃不辣的菜都觉得味道不错。。。）。

吃完饭后，到附近散步，随便买了明天车上的干粮。途中遇到了一个再路口摆摊卖水果的老爷子，价格很实惠（后来从黄龙回来，又光顾了一次）。晚上9点左右，回到旅馆休息。晚上10点左右，一起在我房间吃东西，玩你画我猜（是有多无聊…）。想起上次部门大家一起出去玩已经是一年之前的事了，现在她们也毕业了。晚上12点左右，四姑娘各自回自己房间了。打开平板，打算记录下行程，觉得太累没动力写下去（囧），顺手打开了下载好的《余罪》（恩，此处为我第二天早上睡过头埋下了伏笔…）。

第二天结束。
